# Note App

## Operations
- create category
- create series

- create note
- update note
- change note status (Draft, Publish, Archive, Trash)
- delete note

- view all notes
- view series notes
- view category notes
- view specific note

## Requirements
- python 3.8
- flask 1.0
- sqlite 
- 

## Run following Commands in shell to run the app.
export FLASK_APP=flaskr
export FLASK_ENV=development
flask run
